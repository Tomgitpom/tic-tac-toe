# Project Plan / Projektisuunnitelma

## English

### Project idea
The goal of this project is to create a customizable Tic-Tac-Toe game using Python and Tkinter.

### Main goals
- Create a working graphical game
- Let the user choose board size
- Let the user choose win length
- Add score tracking
- Add optional AI opponent
- Add difficulty levels

### Technologies
- Python
- Tkinter
- GitHub

### Work process
The project started as a basic Tic-Tac-Toe game.  
After that, more features were added:
- custom board size
- custom win length
- score system
- AI opponent
- difficulty buttons
- win highlight

### Future improvements
- stronger AI
- saving scores to database
- cleaner user interface

## Suomi

### Projektin idea
Tämän projektin tavoitteena on tehdä muokattava ristinollapeli Pythonilla ja Tkinterillä.

### Päätavoitteet
- tehdä toimiva graafinen peli
- antaa käyttäjän valita pelilaudan koko
- antaa käyttäjän valita voittopituus
- lisätä pisteiden seuranta
- lisätä mahdollisuus pelata AI:ta vastaan
- lisätä vaikeustasot

### Teknologiat
- Python
- Tkinter
- GitHub

### Työprosessi
Projekti alkoi tavallisena ristinollapelinä.  
Sen jälkeen peliin lisättiin uusia ominaisuuksia:
- pelilaudan koko
- voittopituus
- pistejärjestelmä
- AI-vastustaja
- vaikeustason valintanapit
- voittorivin korostus

### Jatkokehitys
- parempi AI
- pisteiden tallennus tietokantaan
- siistimpi käyttöliittymä
